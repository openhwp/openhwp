## 1. 개요

HWPML은 한글 워드 프로세서 문서를 기술하기 위한 **W3C XML 기반의 개방형 마크업 언어**이다.

## 2. 형식 설명

### 2.1. 기본 속성 값 형식 설명

엘리먼트 설명 테이블 형식:

| 항목 | 설명 |
|------|------|
| 엘리먼트 명 | 엘리먼트 이름 |
| 설명 | 엘리먼트에 대한 설명 |
| 부모 엘리먼트 | 상위 엘리먼트 |
| 자식 엘리먼트/엘리먼트 값 | 하위 엘리먼트 또는 값 |
| 속성 | 속성명, 설명, 값의 범위, 기본값 |

**속성값 표기 규칙:**
- 열거형: 값들을 `|`으로 구분 (예: `Left | Right | Center`)
- 범위: `최소값 ~ 최대값` 형식 (예: `0 ~ 100`)
- 서술식: 일반적인 값의 범위를 문장으로 표기

## 3. 루트 엘리먼트

### HWPML

**설명:** HWPML의 시작을 알리는 루트 엘리먼트

**부모 엘리먼트:** 없음

**자식 엘리먼트:** `HEAD`, `BODY`, `TAIL`

**속성:**

| 속성명 | 설명 | 값의 범위 | 기본값 |
|--------|------|-----------|--------|
| `Version` | HWPML의 버전 | | `2.8` |
| `SubVersion` | | | `8.0.0.0` |
| `Style` | | `embed \| export` | `embed` |

**예제:**

```xml
<HWPML Version="2.8" SubVersion="8.0.0.0" Style="embed">
  <HEAD>...</HEAD>
  <BODY>...</BODY>
  <TAIL>...</TAIL>
</HWPML>
```

## 4. 헤더 엘리먼트

### 4.1. HEAD

**설명:** 문서요약정보 외에 글꼴, 글자속성, 문단속성에 대한 테이블 등 문서에 대한 전반적인 정보를 담고 있다.

**부모 엘리먼트:** `HWPML`

**자식 엘리먼트:** `DOCSUMMARY`, `DOCSETTING`, `MAPPINGTABLE`

**속성:**

| 속성명 | 설명 | 값의 범위 | 기본값 |
|--------|------|-----------|--------|
| `SecCnt` | 구역의 개수 | 0 이상의 정수형 | |

### 4.2. 문서 요약 정보 엘리먼트

#### DOCSUMMARY

**설명:** 문서요약정보를 담는다.

**부모 엘리먼트:** `HEAD`

**자식 엘리먼트:** `TITLE`, `SUBJECT`, `AUTHOR`, `DATE`, `KEYWORDS`, `COMMENTS`, `FORBIDDENSTRING`

#### TITLE

**설명:** 문서 제목

**부모 엘리먼트:** `DOCSUMMARY`

**엘리먼트 값:** 문자열 형태의 문서 제목

#### SUBJECT

**설명:** 문서 주제

**부모 엘리먼트:** `DOCSUMMARY`

**엘리먼트 값:** 문자열 형태의 문서 주제

#### AUTHOR

**설명:** 문서 저자

**부모 엘리먼트:** `DOCSUMMARY`

**엘리먼트 값:** 문자열 형태의 문서 저자

#### DATE

**설명:** 작성 날짜

**부모 엘리먼트:** `DOCSUMMARY`

**엘리먼트 값:** 문자열 형태의 문서 작성 날짜

#### KEYWORDS

**설명:** 키워드

**부모 엘리먼트:** `DOCSUMMARY`

**엘리먼트 값:** 문자열 형태의 문서 키워드

#### COMMENTS

**설명:** 기타 설명

**부모 엘리먼트:** `DOCSUMMARY`

**엘리먼트 값:** 문자열 형태의 기타 설명

#### FORBIDDENSTRING

**설명:** 금칙 문자

**부모 엘리먼트:** `DOCSUMMARY`

**자식 엘리먼트:** `FORBIDDEN`

#### FORBIDDEN

**부모 엘리먼트:** `FORBIDDENSTRING`

### 4.4. 문서 글꼴/스타일 정보

#### MAPPINGTABLE

**설명:** 본문에 사용 중인 글꼴, 글자속성, 문단속성, 탭, 스타일 등등에 대한 세부정보를 담고 있다.

**부모 엘리먼트:** `HEAD`

**자식 엘리먼트:** `BINDATALIST`, `FACENAMELIST`, `BORDERFILLLIST`, `CHARSHAPELIST`, `TABDEFLIST`, `NUMBERINGLIST`, `BULLETLIST`, `PARASHAPELIST`, `STYLELIST`, `MEMOSHAPELIST`

#### 4.4.1. 문서 내 그림/OLE 정보

##### BINDATALIST

**설명:** 그림, OLE등의 바이너리 데이터 리스트

**부모 엘리먼트:** `MAPPINGTABLE`

**자식 엘리먼트:** `BINITEM`

**속성:**

| 속성명 | 설명 | 값의 범위 | 기본값 |
|--------|------|-----------|--------|
| `Count` | BINITEM의 갯수 | 0 이상의 정수형 | `0` |

##### BINITEM

**설명:** 그림, OLE등의 바이너리 데이터 아이템에 대한 정보

**부모 엘리먼트:** `BINDATALIST`

**속성:**

| 속성명 | 설명 | 값의 범위 | 기본값 |
|--------|------|-----------|--------|
| `Type` | | `Link \| Embedding \| Storage` | |
| `APath` | Type이 "Link"일때, 연결 파일의 절대 경로 | | |
| `RPath` | Type이 "Link"일때, 연결 파일의 상대 경로 | | |
| `BinData` | Type이 "Embedding"이거나 "Storage"일때, BINDATASTORAGE에 저장된 바이너리 데이터의 아이디 | | |
| `Format` | Type이 "Embedding"일때, 바이너리 데이터의 포맷종류 | `jpg \| bmp \| gif \| ole` | |

**참고:** 
- 그림의 경우 "Link"와 "Embedding"만 가능
- OLE의 경우 "Storage"만 가능
- 그림의 경우 Format은 "jpg", "bmp", "gif" 중 선택
- OLE의 경우 Format은 "ole"만 가능

**설명:** 헤더와 본문정보 외에 기타 정보를 담고 있다.

**부모 엘리먼트:** `HWPML`

**자식 엘리먼트:** `BINDATASTORAGE`, `SCRIPTCODE`, `XMLTEMPLATE`

### 6.2. BINDATASTORAGE

**설명:** 바이너리 데이타 저장소

**부모 엘리먼트:** `TAIL`

**자식 엘리먼트:** `BINDATA`

### 6.3. BINDATA

**설명:** 바이너리 데이타

**부모 엘리먼트:** `BINDATASTORAGE`

**엘리먼트 값:** 문자열

**속성:**

| 속성명 | 설명 | 값의 범위 | 기본값 |
|--------|------|-----------|--------|
| `Id` | 바이너리 데이타 아이디 | 문자열 | |
| `Size` | 바이너리 데이타 크기 | | |
| `Encoding` | 인코딩 방식. Base64로 고정되어 있다 | `Base64` | `Base64` |
| `Compress` | 압축 여부 | `true \| false` | `true` |

### 6.4. SCRIPTCODE

**설명:** 스크립트 코드

**부모 엘리먼트:** `TAIL`

**자식 엘리먼트:** `SCRIPTHEADER`, `SCRIPTSOURCE`, `PRESCRIPT`, `POSTSCRIPT`

**속성:**

| 속성명 | 설명 | 값의 범위 | 기본값 |
|--------|------|-----------|--------|
| `Type` | 스크립트 코드 종류 | `JScript` | `JScript` |
| `Version` | 스크립트 코드 버전 | | |

### 6.5. XMLTEMPLATE

**설명:** XML 템플릿

**부모 엘리먼트:** `TAIL`

**자식 엘리먼트:** `SCHEMA`, `INSTANCE`

### 6.6. SCHEMA

**설명:** XML 스키마

**부모 엘리먼트:** `XMLTEMPLATE`

**엘리먼트 값:** 문자열

### 6.7. INSTANCE

**설명:** XML 인스턴스
